使用Supervisor管理程序

人生苦短，我用Supervisor

首先通过如下命令安装supervisorsudo apt-get install supervisor

安装完成之后，默认的配置文件应该在/etc/supervisor/conf.d/下面，此后你如果想创建一个新的使用supervisor运行的程序，则在此文件夹里面添加配置文件。

这里提供一个参考例子：

[inet_http_server]
port = 127.0.0.1:8011
username = scott
password = yourpassword
```
